# Expense & Budget Visualizer

## Description
A simple web application to track daily expenses and visualize spending using a pie chart.

## Features
- Add and delete transactions
- Display total balance
- Persistent data using Local Storage
- Interactive pie chart visualization
- Dark mode toggle
- Sort transactions (date, amount, category)
- Spending limit alert & highlight

## Technologies
- HTML
- CSS
- JavaScript
- Chart.js

## How to Use
1. Enter item name, amount, and category
2. Click "Add Transaction"
3. View total spending and chart updates automatically
4. Toggle dark mode if needed
